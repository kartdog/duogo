git status (ver cambios)
git add . (confirmar cambios, prepararlos para hacer commit)
git commit -m "" (Subir los cambios/version a GitHub, con mensaje.)

git config --global user.name/email "usuario/email" (configurar autor responsable)

git remote add origin https://github.com/usuario/repositorio.git (definir origin)
git push -u origin main (subir a origin)
